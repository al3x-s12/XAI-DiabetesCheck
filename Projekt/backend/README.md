# Diabetes Risk Prediction API

Backend für die XAI-DiabetesCheck Web-App.

## 🚀 Schnellstart

1. **Umgebung einrichten:**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
```

## Backend starten
```
cd backend
python app.py
```
